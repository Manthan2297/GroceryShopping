# Manthan's Grocery Center E-Commerce Platform

This repository contains the Java code for an e-commerce platform designed for Manthan's Grocery Center. The platform facilitates online grocery shopping, offering a wide range of products, user authentication, cart management, and secure payment processing.

## Features

- **User Authentication**: Secure login and registration system for users.
- **Product Catalog**: Browse through categories and products to find your grocery needs.
- **Shopping Cart**: Manage your shopping cart, adding or removing items as needed.
- **Order Processing**: Review your orders and proceed to checkout.
- **Payment Processing**: Securely process payments with integrated payment methods.

## Technical Stack

- **Java**: Core programming language used for the platform development.
- **MySQL**: Database for storing user data, product information, and order details.
- **JDBC**: Java Database Connectivity for database interaction.

## Setup and Installation

1. Clone the repository to your local machine.
2. Ensure you have Java and MySQL installed.
3. Update the database credentials in `DatabaseConnection.java` to match your MySQL setup.
4. Compile and run the application.

## Usage

After setting up the project, run `MainC.java` to start the application. Follow the on-screen prompts to navigate through the platform.
